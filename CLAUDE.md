## 앱 구조 (페이지/화면)

```
/                      → Dashboard (검사 목록, 완료 상태)
/test/:testId          → 검사 진행 (문항 표시, 응답 수집)
/result/:testId        → 개별 검사 결과 + 맞춤 조언
/profile               → 통합 프로파일 (모든 검사 결과 종합)
/group                 → 그룹 비교 (Supabase 연동, 레이더차트)
```

## 핵심 기능 요구사항

### 1. 검사 진행 (TestRunner)
- 에니어그램: 유형별로 페이지 나누기 (20문항 × 9유형 = 9페이지)
- 나머지 검사: 전체 한 페이지 또는 10문항씩 페이지 나누기
- 각 문항마다 응답하지 않으면 다음 페이지 이동 불가
- 응답 진행률 상단 progress bar로 표시
- 페이지 이동 시 스크롤 상단 이동

### 2. 자동 채점 (Scoring)
- `data/scoring.ts`에 모든 채점 함수 구현
- 역채점 문항 자동 처리 (burnout: 6,7,8번 / ECR: 회피 7문항 전체 / ProQOL: 1,4,15,17,29번)
- 결과를 localStorage에 저장 (세션 간 유지)

### 3. 결과 표시
- **에니어그램**: 상위 3위 유형 점수 바차트, 1위 유형 상세(강점/약점/학생 궁합/팁), 날개 표시, 3가지 하위유형 설명
- **교사 소진**: 전체 소진 수준 (안정/주의/위험/심각), 5개 하위척도 레이더차트, 높은 척도 순서로 맞춤 조언 목록
- **그 외**: 해당 척도별 점수와 해석 텍스트

### 4. 그룹 공유 (Supabase)
- 닉네임만 입력하면 결과 공유 가능 (회원가입 없음)
- 공유된 그룹원 결과를 레이더차트로 비교
- 에니어그램 유형 분포 원형 차트

## 채점 로직 요약

### 에니어그램
```
각 유형 20문항 합산 → 최고점 유형 = 주유형
날개 = 인접 유형(id±1) 중 점수 높은 쪽
결과 표시: Top3 유형 + 주유형 상세 + 날개
```

### 교사 소진
```
역채점: 6,7,8번 → score = 6 - raw
하위척도 점수:
  교권위기감(1-5):      합산 / 25 = 비율
  무능감(6-9):         합산 / 20
  좌절감(10-13):       합산 / 20
  행정업무 부담감(14-17): 합산 / 20
  교직 회의감(18-22):   합산 / 25

수준 기준 (비율):
  0.00-0.44 = low (안정)
  0.45-0.66 = mid (주의)
  0.67-1.00 = high (위험)

총점 = 합산 / 110:
  ≤0.40 = 안정  |  ≤0.55 = 주의  |  ≤0.72 = 위험  |  >0.72 = 심각
```

### 빅5 (BFI-K-10)
```
역채점: 1,3,4,5,7번
각 요인 2문항 평균 (1-5점)
5개 요인: 외향성(1R,6), 친화성(2,7R), 성실성(3R,8), 신경증(4R,9), 개방성(5R,10)
```

### HSP (K-HSPS-18)
```
역채점 없음. 18문항 평균 (1-7점)
≥5.25 = 고민감(난초형) | 4.31-5.24 = 중민감(튤립형) | <4.31 = 저민감(민들레형)
```

### ECR 성인 애착
```
불안(7문항): 평균 직접 계산
회피(7문항): 전부 역채점 후 평균 (1→7, 2→6, ...)
4유형: 안정(불안↓회피↓) | 몰두(불안↑회피↓) | 무시(불안↓회피↑) | 두려움(불안↑회피↑)
중앙값 분할 기준 (4점 = 분기점)
```

### 갈등해결 스타일 (TKI)
```
5개 스타일 × 3문항 평균 (1-5점)
경쟁(1-3), 회피(4-6), 타협(7-9), 협력(10-12), 수용(13-15)
```

### ProQOL
```
역채점: 1,4,15,17,29번
공감만족(CS): 3,6,12,16,18,20,22,24,27,30 → 합산
번아웃(BO): 1R,4R,8,10,15R,17R,19,21,26,29R → 합산
이차외상(STS): 2,5,7,9,11,13,14,23,25,28 → 합산
수준: ≤22=낮음 | 23-41=보통 | ≥42=높음
```

## 주의사항

1. **검사 문항 저작권**: 검사 문항은 교육·연구 목적으로만 사용. 상업적 배포 금지.
2. **데이터 저장**: 기본은 localStorage. 그룹 공유 시만 Supabase 업로드 (닉네임만, 개인정보 없음).
3. **역채점 주의**: burnout 6,7,8번은 긍정문 → 반드시 역채점 처리.
4. **에니어그램 페이지**: 9유형 × 20문항을 유형별로 한 페이지씩 (총 9페이지). 한 페이지에 180문항 모두 표시하면 안 됨.
